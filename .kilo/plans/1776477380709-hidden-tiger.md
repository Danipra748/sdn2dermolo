# Plan: Update Hero CTA Buttons and Section Title

## Objective
1. Update hero section CTA buttons to use proper route URLs instead of anchor links
   - "Tentang Kami" button → route to `/tentang-kami` (route name: `about`)
   - "Hubungi Kami" button → route to `/kontak` (route name: `contact`)
2. Change the section title in the "Tentang Kami" section from "Sambutan Kepala Sekolah" to "Sambutan"

## File to Modify
- `resources/views/spa/partials/home.blade.php`

## Specific Changes

### Change 1: Hero CTA Buttons (lines 79-86 and 123-130)
Replace anchor links with route() helpers:

**Before:**
```blade
<a href="#tentang" class="...">Tentang Kami</a>
<a href="#kontak" class="...">Hubungi Kami</a>
```

**After:**
```blade
<a href="{{ route('about') }}" class="...">Tentang Kami</a>
<a href="{{ route('contact') }}" class="...">Hubungi Kami</a>
```

### Change 2: Section Title (line 164)
Replace the h2 heading text:

**Before:**
```blade
<h2 class="section-title ...">Sambutan Kepala Sekolah</h2>
```

**After:**
```blade
<h2 class="section-title ...">Sambutan</h2>
```
